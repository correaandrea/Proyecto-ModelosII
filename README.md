# House-Rent-Prediction
Proyecto final - Modelos y Simulación de Sistemas II

# Descripción
Este repositorio corresponde al proyecto final del curso Modelos y Simulación de Sistemas 2. Dentro de este se encuentran 2 archivos principales los cuales son: El Informe en formato PDF en el cual se encuentra documentado el proyecto realizado y el otro archivo es el Notebook realizado para el proyecto. Este Notebook comprende todas las fases: la carga de datos, preprocesamiento, entrenamiento, evaluación y reducción de dimensión.
Es posible acceder al notebook desde Google Colab también a través de este link: https://colab.research.google.com/drive/1q5NEtCW4NnYYI4PmiE_HKD05C2DBkeJD?usp=sharing

Se adjunta también el video con la explicación del proyecto desarrollado: https://youtu.be/ZKXx9oDgBaU
